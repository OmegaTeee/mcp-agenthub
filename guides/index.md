# User Guide

This folder is designed to support users after an agent has built the project. It includes AI-assisted image generation configurations for desktop applications and a comparison table for notes.

The it also contains user-facing guides for post-installation configuration, app integrations, and reference materials, such as Figma, ComfyUI, Keychain, LaunchAgent, and more.

Quick links:

- `getting-started.md` — Quick start and verification
- `keychain-setup.md` — Secure credential storage
- `launchagent-setup.md` — Background service setup
- `app-configs.md` — Per-app configuration (Claude, VS Code, Raycast, Obsidian)
- `figma-integration.md` — Design-to-code workflows
- `comfyui-integration.md` — Image generation workflows
- `comparison-table.md` — Comparison notes and decision framework
