# Client Integrations

**Connect your favorite apps to AgentHub**

Configure desktop applications to use AgentHub as their MCP router for enhanced functionality.

## Available Integrations

### Primary Clients (Recommended)

1. **[Claude Desktop](claude-desktop.md)** - Claude's official desktop app
   - DeepSeek-R1 prompt enhancement
   - 7+ MCP servers
   - Structured reasoning workflows
   - **Best for:** Research, content creation, general conversation

2. **[VS Code](vscode.md)** - Visual Studio Code + Extensions
   - Qwen3-Coder enhancement (code-first responses)
   - File operations and refactoring
   - Works with Cline, Claude Code, Continue
   - **Best for:** Software development, code editing

3. **[Raycast](raycast.md)** - macOS launcher and productivity tool
   - Action-oriented CLI commands
   - Concise responses (< 200 words)
   - Quick documentation lookup
   - **Best for:** Quick commands, productivity workflows

### Specialized Integrations

4. **[Figma](figma.md)** - Design tool integration
   - Design-to-code workflows
   - Component extraction
   - **Best for:** Designers converting designs to code

5. **[ComfyUI](comfyui.md)** - Image generation workflows
   - AI image generation
   - Workflow automation
   - **Best for:** Creative AI workflows

### Quick Reference

6. **[Quick Config](quick-config.md)** - Configuration snippets for all clients
   - Copy-paste configs
   - Side-by-side comparison
   - Troubleshooting tips

## Which Integration Should I Use?

| Use Case | Recommended Client |
|----------|-------------------|
| Research & writing | Claude Desktop |
| Software development | VS Code |
| Quick commands | Raycast |
| Design work | Figma + Claude Desktop |
| AI art generation | ComfyUI |

**Pro tip:** You can use multiple integrations simultaneously! Each client gets custom prompt enhancement based on its use case.
